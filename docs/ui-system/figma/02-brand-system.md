# Brand System

- **Logo:** radar sweep + emerald signal-ping. Lockups: full-color / mono-light / mono-dark. Clear space = height of "D". Min 96px lockup / 16px icon.
- **Colors:** Deep Navy #0B1F3A · Electric Blue #2563EB · Emerald #10B981 + semantic accents. Full neutral scales per theme.
- **Type:** Manrope (display) · Inter (body) · JetBrains Mono (numeric).
- **Voice:** trustworthy, intelligent, analytical, premium, educational. No hype, no advice, no emoji.
- **Do/Don't:** never recolor/rotate/stretch the mark; never a signal without a reason; numbers always mono.
