# Mobile screens

8 screens × 3 platforms (pages 10–12): Dashboard, Recommendations, Watchlist, Portfolio, AI Assistant, Alerts, News, Settings.
- **iOS:** Dynamic Island, large titles, segmented controls, grouped inset lists, blurred tab bar + home indicator, swipe actions, sheets.
- **Android:** Material 3 — top app bar, filter chips, FAB, pill bottom nav, Roboto, tonal cards.
- **PWA:** install banner, offline-aware, pull-to-refresh, brand status bar.
Designed natively per platform — not shrunk desktop.
