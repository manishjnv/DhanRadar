# Retention Framework

## Definitions
- **Active:** performed a meaningful action (research/AI/portfolio/alert) that day.
- **Retention:** D1/D7/D30 classic + **unbounded** (returned in week N); rolling (any action in trailing 7/28d).
- **Stickiness:** DAU/MAU.

## Cohorts & curves
- Cohort by signup week; retention curve target: flatten by D30 (not zero) — a healthy "smile" for power users.
- Segment by persona (starter/picker/pro/learner), acquisition channel, activation type.

## Resurrection & churn
- Dormant (no action 14d) → resurrection campaigns (digest, holding-moved, score-change).
- Churn (paying→canceled / 30d inactive) → exit survey + win-back.
- **Leading churn indicators:** alert fatigue (too many → mute), zero AI use, single-session users, no watchlist.

## Habit loops (drive retention)
- Watchlist → alert → re-evaluate (the core loop).
- Daily market pulse + learning streak.
- Monthly portfolio health report.

## Metrics
- D7/D30 retention by cohort, DAU/MAU, actions/active-user, retention by feature-adopted (watchlist adopters retain X% better → drive adoption).
