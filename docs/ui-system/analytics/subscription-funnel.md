# Subscription Funnel

## Funnel stages
```
visit → signup_completed → activation_completed → engaged(D7) →
  paywall_hit → pricing_viewed → plan_selected → checkout_started →
  checkout_succeeded (PAYING) → retained_30d → renewed
```

## Conversion targets (year-2)
| Step | Target |
|---|---|
| visit → signup | 8–12% |
| signup → activation | ≥ 60% |
| activation → paid (90d) | 4–6% |
| trial → paid | ≥ 35% |
| checkout_started → succeeded | ≥ 85% |
| monthly churn | < 4% |

## Levers (by stage)
- **Signup:** SEO public score pages; no-card trial; social auth.
- **Activation:** 60-second cold-start card; pre-filled watchlist.
- **Paid:** contextual paywall at demonstrated-value moment (fair value, unlimited AI, screener save); annual −20%.
- **Retention:** alerts that matter, monthly portfolio health, learning streak.
- **Win-back:** dormant → digest + "your portfolio moved" nudges.

## Analysis
- Cohort by signup week; segment by acquisition channel, activation-via, persona.
- Drop-off diagnosis per step; A/B on paywall copy/placement, trial length, annual discount.
