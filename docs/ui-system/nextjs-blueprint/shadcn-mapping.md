| DhanRadar | shadcn base |
|---|---|
| Button | button |
| Input/Select/Switch/Slider | input/select/switch/slider |
| Card | card |
| Tabs | tabs |
| Dialog/Sheet | dialog/sheet |
| Popover/Tooltip | popover/tooltip |
| Table | table (+ @tanstack/table for sort/virtual) |
