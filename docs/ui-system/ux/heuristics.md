# UX Heuristics (Nielsen, applied)

1. **Visibility of status** — loading skeletons, sync states, confidence on every AI answer.
2. **Match to real world** — ₹/INR conventions, plain-English metrics, Indian instruments.
3. **User control** — undo on destructive actions, cancel anytime, no dark patterns.
4. **Consistency** — one design system; signal→color→label mapping identical everywhere.
5. **Error prevention** — conservative alert defaults, idempotent payments, validation before submit.
6. **Recognition over recall** — search-first, recent/suggested, persistent nav.
7. **Flexibility** — ⌘K power-user search; presets for beginners.
8. **Minimalist** — data-first, whitespace, no slop; one score answers one question.
9. **Error recovery** — actionable error states with retry + support.
10. **Help** — inline "explain this" on every metric; Methodology one click from any Score.
